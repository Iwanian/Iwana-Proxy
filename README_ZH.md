# ✨ 功能

- 🚀 简洁现代的用户界面
- 🌍 搜索并验证可用代理
- 📡 内置延迟（Ping）测试
- 📊 智能性能排序
- 🌐 按地区筛选代理
- 🔄 自动更新代理列表
- 💻 支持 Windows、Linux、macOS 和 Android
- ❤️ 完全免费且开源

# 🚀 快速开始

1. 从 Releases 页面下载最新版本。
2. 启动应用程序。
3. 选择您喜欢的语言。
4. 点击 Scan Proxies。
5. 选择代理并连接。
