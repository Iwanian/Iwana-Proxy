## ✨ ویژگی‌ها

- رابط کاربری مدرن
- پشتیبانی از چندین نوع پروکسی
- پشتیبانی از Subscription
- افزودن از طریق QR Code
- افزودن از طریق لینک
- تست تأخیر (Latency/Ping)
- مرتب‌سازی هوشمند
- سرورهای موردعلاقه
- فیلتر بر اساس ریجن
- بروزرسانی خودکار
- پشتیبانی از چندین سیستم‌عامل
- متن‌باز
